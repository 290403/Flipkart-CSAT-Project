# Flipkart-CSAT Analysis 
This project focuses on classifying customer satisfaction for Flipkart users based on structured data (like ratings, order details, etc.) using classical machine learning algorithms. The goal is to predict whether a customer is satisfied or unsatisfied, which can help automate feedback analysis and improve support workflows.

 Models Used
Three supervised classification models were trained and compared:

Logistic Regression – a simple and interpretable baseline

Random Forest Classifier – ensemble model with high accuracy

LightGBM Classifier – gradient boosting framework optimized for performance
